# ReBioClim

## Restoring urban streams to promote Biodiversity, Climate adaptation and to improve quality of life in cities
https://www.interreg-central.eu/projects/rebioclim/

This GitHub repository is a collaboration platform for the project ReBioClim regarding the steps of spatial analysis in:
- Data collection
- Delineation
- Calculation of variables per unit
- GNN as a spatial variable as well
- Clustering for typologies
- Calculation of scores for restoration potentials
- Synergies and trade-offs of variables in the restoration potentials
